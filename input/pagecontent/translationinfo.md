# Translation Information

This Implementation Guide is authored in **English** as the primary language and provides translations in **Lithuanian (lt)**.

## Translation Status

The following content is translated:
- Narrative pages (index, profiles, terminology, changelog)
- Resource titles and descriptions

## Contributing Translations

Translations are maintained by the Lithuanian Medical Library. To contribute or report translation issues, please contact:
- Email: info@medicinosnk.lt
- Publisher: [Lithuanian Medical Library](https://medicinosnk.lt)

## Technical Details

- **Default language:** English (en)
- **Supported languages:** Lithuanian (lt)
- **Translation sources:** `input/translations/lt`

For more information about FHIR IG translation mechanisms, see the [FHIR IG Guidance on Multi-Language IGs](https://build.fhir.org/ig/FHIR/ig-guidance/languages.html).
